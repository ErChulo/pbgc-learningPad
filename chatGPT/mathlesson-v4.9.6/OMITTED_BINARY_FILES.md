# Omitted binary files

The MathLesson v4.9.6 package was uploaded through the ChatGPT GitHub connector as extracted UTF-8 text files, not as a ZIP archive.

The following binary MP4 files from the local ZIP package were intentionally omitted because the current connector write path is text-file oriented and does not safely upload binary files as normal repository contents:

- `OpeningManim-1.mp4`
- `SineCurveUnitCircle-1.mp4`
- `MovingAngle-1.mp4`

To restore full parity with the local ZIP package, add those three MP4 files locally under this same folder and push them with normal `git` commands.
