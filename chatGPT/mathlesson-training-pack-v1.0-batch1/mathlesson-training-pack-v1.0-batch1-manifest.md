# MathLesson Training Pack — v1.0 Batch 1 Manifest

## Files

- `feature-tour-basic.json` — basic feature tour and common authoring blocks.
- `mermaid-lab.json` — Mermaid syntax safety and corrected-import checks.
- `export-layout-lab.json` — Print/PDF and Reveal layout controls.

## Required app baseline

- `mathlesson.v4.7.1.html`

## Batch 1 acceptance checks

For each lesson:

1. Import JSON successfully.
2. Confirm lesson title is correct.
3. Render all sections.
4. Run Prototype Audit.
5. Export Active Lesson JSON and re-import it.
6. Export Student Lesson + Quiz where applicable.
7. Confirm quiz Submit / Recheck works.
8. Confirm inline exercise widgets work where present.
9. Confirm no live intentionally broken Mermaid syntax appears.
10. Record any action that was hard to find in the UI.
