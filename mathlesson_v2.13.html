Authoring Guide

Everything you need to build your own lessons in this file.

How to Switch Lessons

Find this line near the top of the <script type="module"> block:

const ACTIVE_LESSON = "demo"; // "demo" | "bayes" | "annuity" | "calculus"

Change the value to load a different built-in lesson. To add your own, copy any lesson object in the LESSONS map and give it a new key.

Lesson Object Shape

{
  title: "My Lesson",
  audio: { src: "", label: "Narration" },  // src:"" disables audio
  sections: [ /* section objects */ ],
  quiz:     [ /* question objects */ ]
}

Writing Section Content

Each section has an id, a sidebar label, and a contentId pointing to a <template> tag in the HTML. This is the key design decision: content lives in HTML, not in JavaScript strings. LaTeX works exactly as you write it.

<!-- In the CONTENT TEMPLATES block: -->
<template id="my-intro">
  <h1>My Section</h1>
  <p>Inline math: \( x^2 + y^2 = r^2 \)</p>
  <div class="math-display">
    $$\int_a^b f(x)\,dx = F(b) - F(a)$$
  </div>
</template>

Then in the lesson config:

sections: [
  { id: "intro", label: "§1  Introduction", contentId: "my-intro" }
]

Callout Boxes

Three styles — add them directly in your <template>:

<div class="callout def">
  <div class="callout-tag">Definition 1.1</div>
  Content with \( math \).
</div>

<div class="callout thm">...</div>  <!-- purple -->
<div class="callout ex">...</div>   <!-- gold -->

Plotly Charts

Add a div with data-plot="key" in your template, then define the plot in the lesson's plots object:

<div data-plot="my-chart" style="height:300px;"></div>

// In lesson config:
plots: {
  "my-chart": (L) => Plotly.newPlot("my-chart",
    [{ x:[1,2,3], y:[1,4,9], type:"scatter" }],
    { ...L }  // L is the shared dark layout base
  )
}

Manim CE Video Slots

<div class="manim-slot">
  <div class="manim-header">scene.mp4 — Title</div>
  <video controls src="scene.mp4"></video>
</div>

Leave src="" to show a placeholder until the video is ready.

Quiz Question Types

All math in stems and feedback is written as normal LaTeX — single backslashes, standard delimiters. No escaping needed.

Multiple Choice — mc

{
  type: "mc",
  stem: "Which equals \( e^{i\pi} \)?",
  options: ["0", "-1", "1", "i"],
  answer: 1,            // zero-based index
  feedback: "By Euler's formula: \( e^{i\pi} = -1 \)."
}

True / False — tf

{
  type: "tf",
  stem: "Every continuous function on \([a,b]\) is Riemann integrable.",
  answer: true,
  feedback: "Continuity on a closed interval guarantees integrability."
}

Symbolic Math Input — math

The student types into a MathLive field. CortexJS Compute Engine grades symbolically — so x+1 and 1+x both pass.

{
  type: "math",
  stem: "Compute \(\frac{d}{dx}\sin x =\)",
  answer: "\\cos(x)",   // LaTeX string compared symbolically
  numeric: false,       // true for pure numeric answers
  hint: "The derivative of sine is cosine.",
  feedback: "$ \frac{d}{dx}\sin x = \cos x $."
}

Free Response — free

{
  type: "free",
  stem: "Prove \(\sqrt{2}\) is irrational.",
  feedback: "Assume \(\sqrt{2}=p/q\) in lowest terms…"
}

Audio Narration

audio: { src: "narration.mp3", label: "Lesson 1 Narration" }

Set src: "" to hide the player. Requires a local server to load audio files.

Running Locally

python -m http.server 8080
# then open http://localhost:8080

You can open the file directly from disk (file://) for everything except audio and external video files.
