Authoring Guide

MathLesson v3.3 — reference for building and using lessons.

Quick Start

Click ✏️ Lesson Editor in the sidebar to open the editor. Pick an example from the dropdown, or build your own. No HTML editing required for user lessons.

Math Syntax — all four styles work everywhere

SyntaxType
$...$Inline
$$...$$Display block
\(...\)Inline
\[...\]Display block

Heads-up: Since $ triggers math, write literal dollar amounts as \$24{,}000.

Editor — ① Sections Tab

Content is HTML with embedded LaTeX. The live preview on the right renders as you type.

Exploration Widgets

2D Interactive Plot
<div class="explore-widget" data-explore="2d"
  data-title="My plot title"
  data-params='[
    {"id":"A","label":"Amplitude","min":0.1,"max":3,"step":0.1,"value":1},
    {"id":"f","label":"Frequency","min":0.5,"max":5,"step":0.5,"value":1}
  ]'
  data-fn="(x,p) => p.A * Math.sin(2*Math.PI*p.f*x)"
  data-xrange="[-2,2]"
  data-xlabel="x" data-ylabel="y"
  style="height:260px;">
</div>
3D Surface Plot
<div class="explore-widget" data-explore="3d"
  data-title="Surface title"
  data-params='[{"id":"f","label":"Frequency","min":1,"max":5,"step":1,"value":2}]'
  data-fn="(x,y,p) => Math.sin(p.f*x)*Math.cos(p.f*y)"
  data-xrange="[-3,3]" data-yrange="[-3,3]"
  style="height:320px;">
</div>
Inline Exercise (symbolic grading)
<div class="exercise-widget"
  data-answer="\frac{n(n+1)}{2}"
  data-hint="Write S twice, add.">
  <div class="exercise-prompt">
    Compute $\sum_{k=1}^n k$.
  </div>
  <div class="exercise-input-row">
    <math-field class="exr-field" placeholder="Your answer…"></math-field>
    <button class="exercise-check-btn">Check</button>
  </div>
  <div class="exercise-result"></div>
  <div class="exercise-hint"></div>
</div>

After 2 wrong attempts the hint appears. Graded by CortexJS Compute Engine (symbolic + numeric fallback).

Editor — ③ Settings Tab

Audio src: Path to an .mp3 relative to this file. Player appears in the top bar.

Export JSON: Downloads your lesson — use for backup or sharing.

Import JSON: Loads a previously exported lesson into the editor.
